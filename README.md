## Example input
> File: perc_init.txt
```python
L T p0 pk dp
100 100 0.1 0.95 0.05
```

**Where:**

- L: the linear size of the system
- T: number of trials
- p0: minimum value ofpin the loop
- pk: maximum value ofpin the loop
- dp: step with whichpchanges inside of the loop
